# skystore
A simple Node.js + MongoDB e-commerce platform with:
- Product listing
- Image upload
- Shopping cart

## Setup
1. Install dependencies:
```bash
npm install
```
2. Run MongoDB locally or connect to MongoDB Atlas.
3. Start server:
```bash
npm run dev
```

API Endpoints:
- GET `/api/products`
- POST `/api/products` (multipart form-data with `image`)
- GET `/api/cart`
- POST `/api/cart`
